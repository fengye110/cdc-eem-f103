very basic cdc-eem implementation with UIP stack for stm32.
Now only arp and icmp echo reply implemented.

For debugging under windows use cdc-eem driver developed by Thesycon (http://www.thesycon.de/eng/usb_network.shtml)
Driver (USB CDC EEM DRIVER hid_global_usb_cdc_eem_driver_v2.51.0.0.zip) found in:
 https://www.hidglobal.com/drivers
 https://www.hidglobal.ru/drivers
